ISO 11452-2

Description
The International Organization for Standardization (ISO) 11452-2 is a technical standard for the design, manufacture, and use of electronic devices and components for passenger cars and commercial vehicles.

Calibration
  • Calibrate with sensor (Narda 604) and store both polarizations in reference file
  • The table for calibration should be empty, but prepared for test.

Test
  • Disable sensor monitoring in the template
  • Select reference file
  • Set the dwell time
