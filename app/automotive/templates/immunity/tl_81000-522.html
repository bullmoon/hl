5.2.2 BCI testing

Unless otherwise specified below, the requirements as per ISO 11452-4 apply to BCI testing.
Only the substitution method is permitted for BCI testing.
Frequency range - 0.1-400 MHz		 Deviating from ISO 11452-4, the test wiring harness must have a length
between 1 700 mm and 2 000 mm.

The minimum dwell time is 2 s.

If the DUT reacts more slowly to interference, then the dwell time must be increased accordingly.

    The tests must be performed in:
  • common mode (ground cable in the current injection probe)
    • and
  • differential mode (ground cable outside of the current injection probe).
    When a DUT has multiple connectors, the test should be repeated with the injection probe clamped around each separate connector bundle, if applicable. Detailed connectors configuration to be tested shall be defined in the test plan and recorded in the test report.

Increments
Testing must preferably be performed using a linear increment. The maximum frequency increments listed in table 6 apply here.
A logarithmic increment may also be used as an alternative (see table 7). This must be agreed upon with the purchaser.

Function Performance Status Classification (FPSC)
    The following status definitions as specified in ISO 11452-1 are used:
  • Status I - The function behaves as specified before, during, and after the test
  • Status II - The function does not behave as specified during the test, but automatically returns to normal operation after the test
    The disturbance (level) up to which this status must be met:
  • L1 - Disturbance level up to which status I must be met
  • L2 - Disturbance level up to which at least status II must be met (status I is permissible as well)
    Based on how customers are affected, there are three categories of effects:
  • Category 1 (A) - Minor effects or negligible malfunctions of the DUT
  • Category 2 (B) - Effects or malfunctions of the DUT with an impact on comfort
  • Category 3 (C) - All significant effects or other effects and malfunctions of the DUT that do not fall into category 1 or category 2
    The difference in test severity levels between each of the three categories is 4 dB.
