5.3.6 Interference emission (Antenna method)

Antenna systems:
  • rod antenna test setup (9 kHz - 30 MHz)
  • biconical antenna test setup (30 - 200 MHz)
  • log periodic antenna test setup (200 - 1000 MHz)
  • horn antenna test setup (1000 - 6000 MHz)
    		•
      Test setup:
    • The test setup is described in CISPR 25.

      In principle, measurements may be conducted with the peak detector in all ranges.
      If the peak measured value lies below the quasi-peak limit, the quasi-peak limit is fulfilled.
